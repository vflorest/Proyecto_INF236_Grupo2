
Para ejecutar el codigo siga los siguientes pasos:

1:Abra 2 terminales en el directorio donde esta el codigo (Hito 6) o navegue hasta el directorio.
3:Levante la api con el siguiente comando en una de las terminales: python .\apy.py
4:Ejecute la interfaz de streamliet con el comando en la otra terminal: streamlit run streamlit_interface.py

Con eso ya se puede usar el programa.
Para mas detalles sobre su uso se puede ver el video demostrativo del hito 5,6,7.
